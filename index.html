Javascript

JavaScript adalah bahasa pemrograman tingkat tinggi yang sering digunakan untuk mengembangkan aplikasi web interaktif dan dinamis. Ini adalah salah satu teknologi utama di web modern yang memungkinkan Anda menambahkan berbagai fitur interaktif, validasi data, animasi, dan banyak lagi ke situs web Anda.

Laravel Monolite

Laravel monolit adalah pendekatan pengembangan web di mana seluruh aplikasi web dibangun dalam satu monolit besar, termasuk semua komponen seperti tampilan, bisnis logic, dan basis data dalam satu kode basis tunggal.

Laravel RestApi

Laravel Resful API adalah kemampuan dalam framework Laravel yang memungkinkan Anda untuk dengan mudah membuat dan mengelola API (Application Programming Interface) yang mengikuti prinsip-prinsip RESTful (Representational State Transfer) untuk berkomunikasi dengan aplikasi lain melalui HTTP.

PHP

PHP adalah bahasa pemrograman sumber terbuka yang sering digunakan untuk pengembangan web. Itu adalah singkatan dari "PHP: Hypertext Preprocessor" dan digunakan untuk membuat situs web dinamis dengan kemampuan untuk berinteraksi dengan basis data dan menghasilkan konten berbasis web secara dinamis.

Vue Js

Vue.js adalah sebuah kerangka kerja JavaScript yang bersifat open-source yang digunakan untuk mengembangkan antarmuka pengguna (UI) pada aplikasi web. Vue.js sering digunakan untuk membangun aplikasi web yang dinamis dan responsif. Ia menawarkan komponen-komponen yang dapat digunakan kembali dan alat-alat untuk mengelola keadaan aplikasi (state), sehingga mempermudah pengembangan aplikasi web yang kompleks. Vue.js sangat fleksibel dan mudah diintegrasikan dengan berbagai proyek, sehingga banyak pengembang yang memilihnya sebagai pilihan mereka dalam mengembangkan antarmuka pengguna aplikasi web.

Express js

Express.js adalah kerangka kerja (framework) Node.js yang bersifat open-source yang digunakan untuk mengembangkan aplikasi web dan aplikasi berbasis server. Express.js menyederhanakan proses pengembangan aplikasi web dengan menyediakan berbagai fitur dan alat yang memungkinkan pengembang untuk dengan mudah mengelola rute, permintaan HTTP, tanggapan, dan berbagai fungsi lain yang dibutuhkan dalam pengembangan aplikasi web.
